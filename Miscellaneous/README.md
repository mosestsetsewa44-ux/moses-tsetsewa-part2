## Project Description

This website is built to represent the Noordelikes Rugby Club, providing users with easy access to information about the club, its history, teams, and contact details. Visitors can also submit an enquiry to join the club through an interactive form. The website is fully responsive and optimized for both desktop and mobile devices.

### Pages Included:
- **Home**: Welcome message, club overview, and navigation links.
- **About Us**: Club history, mission, and team introduction.
- **Teams**: Information on all the club's teams, including junior, senior, women's, and veterans.
- **Contact**: Contact details and a Google Maps integration to help visitors find the club.
- **Enquiry**: A form for prospective members to join the club.

# Changelog

## [1.0.0] - 2025-11-19
### Initial Release
- Created the initial version of the Noordelikes Rugby Club website.
- Implemented basic structure with HTML and CSS.
- Included pages: Home, About Us, Teams, Contact, Enquiry.
- Added responsive layout for desktop and mobile devices.

### Features:
- Navigation bar with links to all pages.
- Hero section with a welcome message.
- Footer with social media icons and copyright information.
- Contact form with basic styling.
- About Us page with club history, mission, and team information.

## [1.1.0] - 2025-11-20
### Improvements:
- Added responsive CSS for tablet and mobile views using Media Queries.
- Enhanced form styling and input validation for the enquiry form.
- Updated `robots.txt` to improve SEO and control crawler access.
- Created the **Sitemap** and integrated it into the website.
- Improved footer and updated social media links.
- Added "Become a Member" call-to-action (CTA) button on the homepage.

### Bug Fixes:
- Fixed alignment issues with the hero section on smaller screens.
- Corrected navigation bar styles on mobile devices.

## [1.2.0] - 2025-11-21
### New Features:
- Implemented JavaScript features such as interactive map (Google Maps).
- Added Gallery lightbox feature.
- Integrated real-time content fetching for team data using JavaScript.
- Added search feature to improve navigation on team and contact pages.
- Enhanced SEO by adding title tags and meta tags for better search engine optimization.
- Refined the "Join Us" page with improved form fields and submission processing.

### Documentation:
- Updated `README.md` with detailed instructions on setup, usage, and contributing.
- Added a full **Changelog** to track the development process.
